stl中主要有6种组件：容器、类属算法、迭代器、函数对象（仿函数）、适配器、分配器。

容器

容器是存储其他对象的集合的对象。stl中有两类容器：序列容器和关联容器

序列容器

主要时下面几种，用得比较多

  1. vector
  2. deque
  3. list

    有序关联容器

  4. set 每个key只能有一个
  5. multi 可以有重复的key
  6. map 每个key只能有一个
  7. multi 可以有重复的key

map的成员函数find,检查name是否是map的一个key，如果是，则返回对应的迭代器，如果不存在，则返回map.end()(序列末尾的后一个位置的迭代器);

此时的输出结果是
2
0
3
说明调用map的[]运算符相当于插入了一个相应类型的默认值的键值对。

类属算法

线面将简单的介绍几个类属算法

find类属算法

where = find(first,last,value)

与map的成员函数类似，find类属算法也是在找到的情况下返回被查找元素的第一次出现的位置的迭代器，没有找到也是返回该容器末尾的后一个位置的迭代器。

merge

merge(first1,last1,first2,last2,result);
将两个升序的序列合并成一个新的升序排列的序列

迭代器

迭代器是用来链接容器和类属算法的，容器提供了可以插入到算法中的迭代器，而类属算法的参数（接口）便是迭代器

由于不同的容器的底层实现原理不同，一种类属算法不可能使用于所有的容器，算法性能上也难以满足，所有将迭代器分类是针对这些问题的主要解决方案。

函数对象(仿函数)

实际上只是重载了operator()操作符的类的对象，相较于回调函数或者函数指针，它可以携带更多的信息等优势

适配器

用来改变其他组件接口的组件成为适配器
如reverse_iterator可以将一个迭代器转换成逆向的迭代器
书中给了一个例子:
由于舍入误差和溢出误差，当一些极小或者极大的数，从小到大累加或者从大到小累加会影响精确性

结果：
1
1.000000119

分配器

stl容器都使用了一种分配器，用来封装内存分配模式的信息，默认的分配器一般满足编程需要，所以书中不会在详细解释。

积累

  • 所有的stl容器都重载了 == 运算符
  • 所有的迭代器都重载了++ 和– 运算符，但有些没有重载+运算符。
